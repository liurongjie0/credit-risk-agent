# 紧急情况处置流程

## 求援电话（中国大陆）

- 120 医疗急救 / 110 报警（山地救援通常由警方协调）
- 拨打时报告：所在线路名、最近的地标或里程碑、伤情、队伍人数。

## 常见情况处置

### 中暑
移至阴凉处，解开衣物，小口补水（含电解质更佳）；意识模糊立即求援。

### 失温
换下湿衣物，用睡袋/急救毯包裹，补充热糖水；禁止摩擦四肢或饮酒。

### 骨折或无法行走
不要移动伤员，固定伤处，保暖并求援；记录时间与位置。

### 蛇虫咬伤
记住咬伤生物外形（拍照），保持伤肢低于心脏，尽快就医；不要用嘴吸出毒液。

## 求援信息模板

```
线路：<线路名>（如 徽杭古道）
位置：<最近地标>，约 <里程> 处
情况：<人数> 人，<伤情描述>
联系：<手机号>，电量 <百分比>
```
